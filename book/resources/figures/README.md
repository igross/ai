# Figures

Purpose
- Final and near-final figures for the manuscript and chapter drafts.

What goes here
- Exported images (e.g., `.png`, `.pdf`, `.svg`) that are referenced in text.
- A short note next to a figure if special handling is needed.

Conventions
- Name files with chapter or section prefix, e.g., `ch03_kpi_tradeoff.png`.
- Source code that generates figures lives in `code/`.
