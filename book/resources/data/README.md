# Data

Purpose
- Data used to support claims, figures, or worked examples in the book.

What goes here
- Raw data files and processed extracts.
- Data dictionaries or README files for datasets.

Conventions
- Prefer small, clean extracts over huge dumps.
- Document source and access date in a sidecar `SOURCE.md` or README.
- If data should not be committed, note that in a local `.gitignore` entry.
