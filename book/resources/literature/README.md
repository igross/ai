# Literature

Purpose
- Style reference and verified sources for the book.

What is here
- `How To Read Numbers V4.pdf` (Chivers & Chivers, 2021)
- `Everything is Predictable V4.pdf` (Tom Chivers, 2024)
- `The AI Does Not Hate You V3.pdf` (Tom Chivers, 2019)

Why these PDFs
- These are here primarily to learn and mirror the writing style and tone.
- They are not primarily for quoting.
- PDFs are slower to search; if Word versions become available, add them here alongside the PDFs.

Usage
- We may ask for examples of phrasing, openings, or stylistic patterns from these PDFs.
- Keep any extracted notes in `STYLE_NOTES.md`.
