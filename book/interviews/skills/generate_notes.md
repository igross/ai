# Skill: Generate Notes (Pass 1 + Pass 2)

## Input
- Cleaned/chunked transcript files.
- Interview packet (`interview_packet.md`) and metadata.

## Output
- `outputs/notes_pass1_index.md`
- `outputs/notes_pass2_booknotes.md`

## Rules
- Pass 1 is descriptive and time-coded, not argumentative.
- Pass 2 is editorial and book-oriented (thesis, anecdotes, frameworks, tensions, chapter links).
- Distinguish evidence from inference.
- Do not fabricate external facts; add unknowns to fact-check list.

## Pass 1 Focus
- Fast outline by timestamp.
- Theme map.
- "Best moments" shortlist.

## Pass 2 Focus
- What the chapter argument gains/loses from this interview.
- Narrative assets (scene/anecdote potential).
- Mechanisms/framework language.
- Productive tensions or contradictions to use honestly.
