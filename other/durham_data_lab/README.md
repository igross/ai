## What this project is

Data Lab is an independent collaboration delivering data/statistics training and business consultancy.
This repository is the shared working space for the team.

## Collaboration Workflow (Codex + GitHub)

1. `main` is protected and always stable.
2. Each person creates a short-lived branch for each task:
   - `david/<task>`
   - `johnpaul/<task>`
   - `demid/<task>`
3. Open a pull request (PR) into `main`.
4. One teammate reviews and approves.
5. Merge PR; delete merged branch.

## File Management Rules

- Keep work inside this project folder only.
- Use `09_shared_workspace/` for current shared drafts and live working files.
- Use `10_final_outputs/` for approved deliverables only (client-ready, agreed versions).
- Keep notes and decisions in `06_meetings_notes/` and `07_actions_decisions/`.
- Do not overwrite someone else's file directly in parallel; instead:
  - edit your branch and open a PR, or
  - create a new dated draft in `09_shared_workspace/`.

## Naming Convention

- Shared drafts: `topic_YYYY-MM-DD_initials.ext`
- Final outputs: `topic_FINAL_YYYY-MM-DD.ext`
