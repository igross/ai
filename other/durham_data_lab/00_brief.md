## Key Concerns / Priorities

### 1) Degree of Durham University involvement
- We want to avoid bureaucracy; too much would stop the project.
- Need a conversation with the relevant person(s) to clarify what we can use and what constraints apply.

### 2) Business Formation & Operations
- If we set up as a business: what type of structure?
- How to start the business.
- How to handle payments.
- Need clear upfront contracts.

### 3) Website
- Need to design and build a website.
- This can be a subproject with its own guidelines.

### 4) Contracts & Tax
- Consider tax implications given we are employed and likely higher-rate (40%).
- Think about payment structures to be tax-efficient.
- Can be an end section / later workstream.

## Team / Governance Aims

- Data Lab brand benefits everyone: good projects raise awareness for all.
