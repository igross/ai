# Project Memory - Tuition Fees

## Scope lock (v0)
- Keep model simple and transparent; no full structural estimation.
- Main comparison: post-2012 UK student loan system vs graduate tax vs general taxation.
- Distributional focus: graduates/non-graduates and heterogeneity by family wealth/income.

## Modeling choices so far
- Baseline framework: simple OLG/lifecycle incidence model.
- Education choice: exogenous in v0.
- Welfare metric: present discounted value (PDV) of net payments/transfers over lifecycle.

## Open questions
- Exact treatment of interest accrual and write-off horizon under current UK rules.
- Whether to model upfront-fee option as endogenous selection by wealthy households or as scenario toggle.
- Which discount rate benchmark(s) to report in baseline comparisons.
