# 08 uk procurement laptop pricing

This project designs and partially implements an empirical UK paper on whether public buyers pay more for standardised laptop SKUs than contemporaneous market benchmarks, and why.

## current status
Design scaffold created: institutional framing, data schemas, identification strategy, writing skeleton, and code stubs.
