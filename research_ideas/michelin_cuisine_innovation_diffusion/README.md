# Michelin cuisine innovation diffusion

## What this project is
This project studies how culinary ideas emerge, diffuse, and potentially homogenize in Michelin-star fine dining over time.

## Current status
Early design phase (pre-paper). A ranked first-pass literature, model, and empirical design is drafted in `notes/05_research_plan.md`.

## Next 3 tasks
1. Finalize scope (geography + years + unit of innovation).
2. Build Michelin restaurant-year panel and chef-network spine.
3. Pilot menu/image measurement on a small city sample.
