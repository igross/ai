# Academic Economics AI Template

This folder is a starter workspace for economists using VS Code with AI coding assistants.

## What this pack is for

Use this as a clean starting point for new research or teaching projects.
It includes reusable skills, agent instructions, and templates.

## Folder structure

- `agents/`:
  role definitions for specialist helper agents.
- `skills/`:
  reusable workflows/checklists and domain instructions.
- `templates/`:
  starter templates (for example referee responses and related documents).
- `memory/`:
  place to keep working notes, assumptions, and prompt patterns.
- `projects/`:
  intentionally blank; create one subfolder per project.

## Quick start (for colleagues)

1. Put this folder on your Desktop or in Documents.
2. Open VS Code.
3. Click `File > Open Folder`.
4. Select this folder.
5. Create your first project inside `projects/`.

## Recommended working pattern

- Keep each project in `projects/<project_name>/`.
- Keep decisions and assumptions in `memory/`.
- Ask AI tools for clear, incremental edits.
- For papers, prefer LaTeX + PDF outputs.

## Sharing notes

If you share this with others, share the entire folder as a single package (zip or folder download) so paths stay intact.
