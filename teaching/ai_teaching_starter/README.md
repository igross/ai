# AI teaching starter packs

This folder prepares two starter distributions for teaching:

- `starter_basic/` for lightweight onboarding
- `starter_advanced/` for full workflow preferences

Recommended publishing model:

1. Create GitHub repo: `davidchivers/ai_install`
2. Publish `starter_basic/` contents to branch `starter_basic`
3. Publish `starter_advanced/` contents to branch `starter_advanced`

Stable ZIP links:

- `https://github.com/davidchivers/ai_install/archive/refs/heads/starter_basic.zip`
- `https://github.com/davidchivers/ai_install/archive/refs/heads/starter_advanced.zip`

This is a starter pack, not a locked system. Users can use it as-is, copy parts, or start a new folder.
