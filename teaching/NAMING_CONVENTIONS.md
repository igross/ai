# Naming Conventions

Use these conventions for readability and consistency:

- Folders: `Title_Case` (use underscores, no spaces), e.g., `AI_Workshop`, `Lecture_Notes`
- Files: `snake_case.md`, e.g., `slides_outline.md`, `session_plan.md`
- Keep names short and descriptive.

## Examples

- Good folder names: `AI_Workshop`, `Referee_Response`, `Data_Cleaning`
- Good file names: `slides_outline.md`, `reading_list.md`, `workshop_agenda.md`
