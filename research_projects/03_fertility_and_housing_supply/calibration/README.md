# Calibration

Use this folder for model-calibration code and run workspaces when calibration starts.

Recommended layout:

- solver source files in this folder
- run workspace subfolder (for example `AI_calibration/`)
- `runtime/data/input/` and `runtime/data/output/` inside run workspace
- `old/` for archived runs

Reference: `_shared/standards/code_calibration_standard.md`
